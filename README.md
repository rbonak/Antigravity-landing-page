# VENORA - Natural Skincare Landing Page

A premium, high-converting landing page built with modern UI/UX design practices, custom micro-interactions, fluid typography, and a 3-layer CSS token architecture.

## Features
- **3-Layer Design Tokens**: Primitive → Semantic → Component tokens for scalability and consistent styling.
- **Custom Cursor**: Sleek custom mouse cursor with magnetic expansion over clickable elements.
- **Micro-Interactions**: Magnetic button pull effects and 3D card tilt animations on hover.
- **Responsive Layout**: CSS Grid Bento Box layout and fluid typography (`clamp()`) adapted for mobile, tablet, and desktop.
- **Accessibility (a11y)**: Full ARIA labeling, semantic markup, and keyboard focus visibility.

## File Structure
- `index.html`: Main HTML5 semantic structure.
- `style.css`: Complete styling system with tokens, reset, typography, and responsive media queries.
- `script.js`: Vanilla JS for intersection observers, parallax scrolling, custom cursor, and magnetic buttons.
- `assets/`: Image assets and avatars.
